Stands for **C**reate, **R**ead, **U**pdate and **D**elete and are the four basic operations of persistent storage.

| CRUD   | HTTP   | SQL    |
| ------ | ------ | ------ |
| Create | POST   | INSERT |
| Read   | GET    | SELECT |
| Update | PUT    | UPDATE |
| Delete | DELETE | DELETE       |
